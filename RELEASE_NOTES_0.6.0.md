# PodPlayer v0.6.0

## 本版重点

- **订阅更新流**：`我的订阅`默认进入按发布时间聚合的更新页，支持节目快速筛选、全部 / 未听完筛选、稳定虚拟列表、单集独立封面、统一回到顶部，以及保留完整节目网格的二级管理入口。
- **单集封面与 Shownotes 阅读**：更新流优先展示单集封面并保留原生外部拖拽；Shownotes 的独立图片改为左对齐、单栏、等比且有稳定高度上限的阅读流。
- **本集 AI 总结**：在用户主动操作、已配置兼容 OpenAI 服务后，可对当前单集文字稿生成总结；同一时间只运行一个总结任务，可取消，切集和删除文稿时会失效旧任务。
- **文字稿一致性**：删除文稿会同步删除该集 AI 精修稿和本集总结，但不会删除音频。部分或取消的 AI 精修不会冒充可用完整结果，重新运行只补缺失段。
- **播放与数据完整性**：收听完成统计覆盖倍速、延迟 tick、自然结束与“本集结束后睡眠”；发现节目去重增加标点归一；睡眠唤醒、输出设备回落与下载续传的既有修复保留。
- **可信 Dev 启动器**：开发环境可显式锁定 worktree、分支、HEAD 与必需祖先；不匹配时 fail closed，并写入运行回执。

## 本地转文字稿与联网边界

- 本地 ASR 和模型一键部署已在 **Windows x64** 验证。安装包不内置约 240 MB 的 SenseVoiceSmall 模型，用户必须在设置页手动部署或选择本地模型目录。
- 模型部署和本地转写不会自动生成文稿、不会上传音频、不会调用联网 AI。
- AI 精修和本集总结均为可选联网操作：只有用户主动点击后，才会把当前文字稿发送到用户配置的兼容 OpenAI endpoint。音频不会上传。
- 远程 endpoint 仅接受 HTTPS；HTTP 只允许 `localhost`、`127.0.0.1` 或 `::1`，避免意外把文本发往不安全的远程 HTTP 服务。
- 本版不包含自动总结、批量总结、说话人分离、全文搜索或“所有 provider 已完成质量认证”等能力。

## 平台与已知限制

- Windows x64 是本版本地 ASR 与模型部署的验证范围。macOS/Linux 仍可使用其它客户端功能，但不宣称本地 ASR 已验证。
- Portable 与 NSIS 安装包当前未做 Authenticode 签名，Windows 首次运行可能出现 SmartScreen 提示。
- 冷加载时单集封面会先显示节目封面，再淡入独立单集封面；该过程是已知体验优化项，不影响播放、拖拽或缓存完整性。
